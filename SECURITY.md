# Security Policy

## Supported versions

ContextBuddy is currently in early versions. Security fixes will be released on the latest version branch.

## Reporting a vulnerability

If you believe you’ve found a security issue, please do **not** open a public GitHub issue.

Instead, email the maintainer with:
- a description of the issue and impact
- minimal reproduction steps
- affected versions (if known)

If you can’t find a contact email, open a GitHub issue with **no sensitive details** and request a private channel.

## Scope notes

ContextBuddy does not transmit data anywhere by default. It runs fully locally unless you explicitly enable optional networked components (e.g., OpenAI embedder).
